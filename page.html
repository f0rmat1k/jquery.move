Examples stand

Simple example

$(this).move({ top: '100px', left: 250 }, 3000);

Example with existing opacity property (hover)

$(this).move({ top: '100px', left: 250 }, 2000);

Example with use '+=' operators

$(this).move({ top: '+=20px', left: '+=30' }, 500);

Example with use '-=' operators (yet broken)

$(this).move({ top: '-=20px', left: '-=30' }, 500);

Example with use percent

$(this).move({ top: '30%', left: '50%' }, 500);
